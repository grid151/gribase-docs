Webhook

Handles post webhook requests from external system.


Keep backwards compatability for Encompass for now, will need to be phased out.
Reads message and headers to validate request.

path Parameters
system
required
string
Enum: "None" "Resware" "AtClose" "Softpro" "eAgency" "Closepin" "Encompass" "X1" "Stacx" "RamQuest" "BlackKnight" "ChatGPT" "ManualEntry" "Inkd"

System to use (will get first integration for this system)

Responses

https://app.gridbase.io/v1/core/webhook/external/{system}
